選び方と政策

9人の有権者がいて、4人の候補者（政策）について、次のように好んでいる。

坂井 (2016), p.43
人数 4人 3人 2人
1位 A C D
2位 B B B
3位 C A C
4位 D D A
  • 多数決
    • Aが最多の4人の支持を得る
  • 決選投票
    • 1回目で2票しか得なかったDが落選
    • 決選投票ではCが5人の支持を得る1
  • ボルダルール：1位に4点、2位に3点、3位に2点、4位に1点を与える
    • Aは24点、Bは27点、Cは24点、Dは15点
